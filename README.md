## Description

API server for [ShortTutoring](https://github.com/amicably-until-the-end/ShortTutoring)

## Specification

ubuntu 22.04.2 LTS   
node.js 18.16.1 LTS

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
